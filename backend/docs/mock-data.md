# Mock Data

Para a demonstração, o sistema é inicializado com os seguintes dados:

-   **Categorias:** Alimentação, Higiene, Limpeza.
-   **Produtos:** Arroz, Feijão, Óleo, Sabão, Detergente.
-   **Cotações:** Histórico dos últimos 3 meses para permitir a visualização de gráficos no frontend.
