# Futuras Melhorias

O SMIM é uma plataforma em constante evolução. Abaixo, listamos os próximos passos planejados:

## 1. Persistência de Dados
-   Migração do Mock para **PostgreSQL** ou **SQLite** utilizando SQLAlchemy ou Tortoise ORM.

## 2. Autenticação e Autorização
-   Implementação de login para administradores via **JWT (JSON Web Tokens)**.
-   Níveis de acesso (Admin vs Consultor).

## 3. Notificações
-   Sistema de alerta por e-mail ou Telegram quando um produto atingir uma variação de preço acima de X%.

## 4. IA e Predição
-   Uso de modelos simples de regressão para prever a inflação do próximo mês baseada na tendência atual.

## 5. Exportação de Relatórios
-   Geração de PDFs e planilhas Excel com o resumo mensal dos preços em Morrinhos.
