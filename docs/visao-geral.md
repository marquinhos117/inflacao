# Visão Geral do Sistema - Inflação Morrinhos-GO

O **Sistema de Monitoramento de Inflação de Morrinhos (SMIM)** é uma plataforma profissional desenvolvida para coletar, analisar e visualizar a variação de preços da cesta básica e outros itens essenciais no município de Morrinhos, Goiás.

## Objetivo
Transformar dados esparsos de preços em informações acionáveis para a comunidade acadêmica, gestores públicos e cidadãos, através de uma interface moderna e APIs robustas.

## Principais Pilares
1.  **Transparência:** Fornecer dados claros sobre a evolução dos preços locais.
2.  **Tecnologia de Ponta:** Utilização de FastAPI para alto desempenho e React para uma experiência de usuário fluida.
3.  **Extensibilidade:** Arquitetura modular que permite a adição de novos indicadores e categorias de produtos.

## Público-Alvo
-   Pesquisadores de economia.
-   Estudantes universitários.
-   Consumidores interessados no poder de compra local.
