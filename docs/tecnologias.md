# Tecnologias Utilizadas

O SMIM utiliza uma stack moderna de "Indústria" para garantir que o projeto seja profissional e atualizado.

## Backend
-   **Linguagem:** Python 3.10+
-   **Framework:** FastAPI
-   **Validação:** Pydantic
-   **Servidor:** Uvicorn
-   **Logs:** Loguru ou Logging nativo estruturado

## Frontend
-   **Linguagem:** TypeScript / JavaScript (ES6+)
-   **Framework:** React
-   **Build Tool:** Vite
-   **Estilização:** Tailwind CSS
-   **Icons:** Lucide React

## Ferramentas de Desenvolvimento
-   **Git:** Controle de versão.
-   **Postman/Insomnia:** Testes de API.
-   **VS Code:** IDE recomendada.
-   **ESLint/Prettier:** Padronização de código.
