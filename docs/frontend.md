# Documentação do Frontend

O frontend do SMIM foi projetado para ser intuitivo, rápido e visualmente impactante.

## Tecnologias Utilizadas
-   **React 18**: Biblioteca base.
-   **Vite**: Ferramenta de build ultra-rápida.
-   **Tailwind CSS**: Framework CSS utilitário para design moderno.
-   **Lucide React**: Biblioteca de ícones elegantes.
-   **Axios**: Cliente HTTP para consumo da API.

## Principais Telas
1.  **Dashboard Home:** Visão geral com cards de indicadores (Índice atual, produtos monitorados, última coleta).
2.  **Gestão de Produtos:** Tabela dinâmica com busca e ações de CRUD.
3.  **Histórico de Preços:** Visualização de variações temporais.
4.  **Simulador de Cesta:** Interface "drag and drop" ou seleção para montar a cesta de consumo.

## Padrões Adotados
-   **Componentização:** Componentes atômicos e reutilizáveis (Buttons, Inputs, Modals).
-   **Responsive Design:** Totalmente funcional em dispositivos móveis e desktops.
-   **Feedback Visual:** Esqueletos de carregamento (Skeletons) e mensagens de erro amigáveis.
