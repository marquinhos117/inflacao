# Componentes do Frontend

A interface é construída com componentes React reutilizáveis.

## UI Base (Shadcn-like)
-   **Card:** Container para informações do dashboard.
-   **Table:** Listagem de produtos e cotações.
-   **Button:** Ações do sistema (Save, Delete, Edit).
-   **Modal:** Formulários de cadastro e edição.

## Componentes de Negócio
-   **InflationChart:** Gráfico de linha mostrando a evolução do índice.
-   **ProductForm:** Formulário dinâmico para produtos.
-   **CestaSimulator:** Interface interativa para simulação de compras.
