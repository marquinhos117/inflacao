# Arquitetura do Frontend

SPA estruturado para escalabilidade.

## Diretórios
-   `/components`: UI Generics.
-   `/pages`: Views principais da aplicação.
-   `/services`: Camada de comunicação com a API (Axios).
-   `/hooks`: Lógica de estado compartilhada.
-   `/context`: Estados globais (ex: Tema, Auth).
